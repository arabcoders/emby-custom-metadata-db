The plugin require simple JSON response with the at least these minimal structure for complete List of possible fields see the the DTO object.
[ { "id": "unique id for example jf-1234", "title": "The Presented title", "matcher": [ "List of possible folder names to match", ], } ]
The url for database reference for example:
https://www.thetvdb.com/?tab=series&id={0}
Notice the {0} it must exists on the url as it will be replaced with given id.
